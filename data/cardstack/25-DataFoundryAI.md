To aid students and researchers getting started with Data Foundry and our local AI, we created skills for AI models to interact with Data Foundry. These skills combined with the extensive documentation written for Data Foundry made it possible for AI to create its own fully-fledged Data Foundry projects. This makes it easier for students and researchers to get started with Data-Enabled Design and takes away the technical burden of creating GDPR compliant data collection probes. 

In This project I was the main responsible developer, working on both the UX and Architecture of these functionalities within the existing Data Foundry platform. Showcasing my skill in implementing and working with AI.
